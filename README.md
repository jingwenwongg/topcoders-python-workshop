# Top Coders Python Workshop

This repository contains my solutions to programming exercises, assessments, and challenge problems completed during the Top Coders Pre-Competition Python Workshop.

The workshop focused on strengthening Python fundamentals, logical thinking, and problem-solving skills through hands-on coding activities and competition-style exercises.

## Topics Covered

* Python Basics
* Decision and Flow Control
* Looping Statements
* Functions
* Sequence Data Structures
* Modules
* Regular Expressions
* Standard Library
* File Handling
* Exception Handling

## Repository Structure

Each topic folder is organized into:

* **i-Design** – guided practice exercises
* **i-Assess** – assessment and challenge questions

## Skills Practiced

* Problem Solving
